world4utauエンジン

このプログラムは森勢将雅氏の「音声分析合成方式 WORLD 0.2.0」
https://github.com/mmorise/World
を用いたUTAU用音声合成エンジンです。
UTAUのresampler.exeの代わりに使ってください．

WORLDの仕組みをよく理解しないで作ってますので、WORLD本来の品質を損なっているかもしれません。

　このプログラムに関する問い合わせを、
　UTAUの作者である飴屋／菖蒲氏にしないで下さい。
　私は飴屋／菖蒲氏とは別人です。

　コードも同様に配布してください。
